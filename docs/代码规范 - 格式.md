# 代码规范 - 格式

1.  方法大括号一律换行显示。

反例：

```java
SetDispEvent(){
    
}
```

正例：

```java
SetDispEvent()
{
    
}
```

1.  以下符号添加空格，美观代码，`=`、 `+`、 `-`、 `*`、 `/`、 `_`、 `:` 左右加空格，`,`之后加空格 。

反例：

```java
GetName(hour,number,year)
{
    s name=“yaoxin”
    s day=hour*number
    s sub=$o(^IRIS("DATE",day,hour,""))
}
```

正例：

```java
GetName(hour, number, year)
{
    s name = “yaoxin”
    s day = hour * number
    s sub = $o(^IRIS("DATE", day, hour, ""))
}
```

1.  方法内命令行均采用一个制表符`tab`缩进，宽度为四个空格宽度。**注意**是一个`tab`键，而不是按四次空格，可在`studio`设置（默认为4）

反例：

```java
GetName(hour,number)
{   
 s name=“yaoxin”
 s day=hour*number
 s sub=$o(^IRIS("DATE",day,hour,""))
}
```

正例：

```java
GetName(hour, number)
{
    s name = “yaoxin”
    s day = hour * number
    s sub = $o(^IRIS("DATE", day, hour, ""))
}
```

1.  `SQL`语句一行`5`个字段。并且换行后`3`个`tab`键为准。逗号在每行行末，不要带入下行开始。因为一行字符书不超过`120`个，即一屏。

注意：`sql`命令全部统一小写。

反例：

```java
ClassMethod InsBox(str As %String) As %Library.String
{
	&SQL(Insert Into DHC_PHBox (
			PHB_No, PHB_Num, PHB_Status, PHB_FLoc_Dr,PHB_TLoc_Dr,PHB_UserCreate_Dr,PHB_DateCreate, PHB_TimeCreate, PHB_PrintFlag, PHB_Remark) 
		values(
			:boxNo, 1, :status, :fromLocID, :toLocID,:userID, :date, :time, "Y", :remark))
	q SQLCODE
}
```

正例 `insert`：

```java
ClassMethod InsBox(str As %String) As %Library.String
{
	&SQL(
		insert into DHC_PHBox 
		(
			PHB_No, PHB_Num, PHB_Status, PHB_FLoc_Dr, PHB_TLoc_Dr,
			PHB_UserCreate_Dr, PHB_DateCreate, PHB_TimeCreate, PHB_PrintFlag, PHB_Remark
		) 
		values
		(
			:boxNo, 1, :status, :fromLocID, :toLocID,
			:userID, :date, :time, "Y", :remark
		)
	)
	q SQLCODE
}
```

正例 `update`：

```java
ClassMethod UpdMenu(arg...) As %Library.String
{
	&sql(
		update CT.IMP.Menu 
         set 
			CIM_Name = :name, CIM_Path = :path, CIM_Component = :component, CIM_Icon = :icon, CIM_OrderNum = :num
        	CIM_Status = :status,
		where 
			ID = :ID
    )
	q SQLCODE
}
```

正例 `delete`：

```java
ClassMethod DelMenu(arg...) As %Library.String
{
	&sql(
		delete BS_PHA_IN.BarCode 
		where 
        	 ID = :ID
	)
	q SQLCODE
}
```

1.  单行字符串拼写最多`5`个字段。因为一行字符书不超过`120`个，即一屏。

反例：

```java
ClassMethod GetString(str As %String) As %Library.String
{
	s str = first _ second _third _ fourth _ fifth _six
}
```

正例：

```java
ClassMethod InsBox(str As %String) As %Library.String
{
	s str = $lb("first", "second", "other", 4, "71")
	s str = str _ $lb("y", "x", "what", "apple", "sum")
	s str = $lts(str, "^")
	w str
}
```

1.  禁止用同一变量后加数字累加。

反例：

```java
ClassMethod GetString(str As %String) As %Library.String
{
    s str1 = first _ second _ third _ fourth _ fifth 
    s str2 = six
    s str = str1 _ str2
}
```

正例：

```java
ClassMethod GetString(str As %String) As %Library.String
{
   s str = first _ second _ third _ fourth _ fifth 
   s str = str _ six
}
```

1.  获取多返回值用数组`%ArrayOfDataTypes`或`JSON`点语法或其他对象数组，不建议使用字符串拼接形式。

反例：

```java
ClassMethod GetPatLocDesc(prescNo)
{
    s str = wardID _ "^" _ wardLocDR _ "^" _ patLocDesc 
	q str
}
```

正例：

```java
ClassMethod GetPatLocDesc(prescNo)
{
    s array = ##class(%ArrayOfDataTypes).%New()
	d array.SetAt(wardID, "wardID")
	d array.SetAt(wardLocDR, "wardLocDR")
	d array.SetAt(patLocDesc, "patLocDesc")
	q array
}
```

正例：

```java
ClassMethod GetPatLocDesc(prescNo)
{
    s obj = {}
    s obj.wardID = wardID
    s obj.wardLocDR = wardLocDR
    s obj.patLocDesc = patLocDesc
	q obj
}
```

1.  禁止系统命名混搭方法中出现`s`和 `set` 统一用命名缩写方式。

反例：

```java
ClassMethod GetPatLocDesc(prescNo)
{
    s str = “wardID”
    set patLocDesc = “病区”
    do ..GetName()
	q str
}
```

正例：

```java
ClassMethod GetPatLocDesc(prescNo)
{
    s str = “wardID”
    s patLocDesc = “病区”
    d ..GetName()
	q str
}
```

1.  禁止命令出现大小写，统一小写。

反例：

```java
ClassMethod GetName(userID)
{
    s name=“yaoxin”
    S age=30
}
```

正例：

```java
ClassMethod GetName(userID)
{
    s name = “yaoxin”
    s age = 30
}
```

1.  尽量使用对仗词。

```java
add/remove
open/close
get/set
start/end
insert/delete
show/hide
lock/unlock
first/last
next/previous
old/new
```

1.  禁止`{}`和`.`同时出现,推荐用使用块级语法，抛弃使用点语法。并且括号要对齐。

反例：

```java
ClassMethod GetName(userID)
{
    i flag="yx" d
    .i other = "other"{
        s name = “other” 
    }
    .s name = "yx"
}
```

正例：

```java
ClassMethod GetName(userID)
{
    if flag="yx" {
        if other = "other" {
            s name = “other” 
        }
        s name = "yx"
    }
}
```

1.  如果`if`语句只有一行，建议使用三元运算符。所有`if`语句都要换行写，即使只有一行。

反例：

```java
i name="yx" s age=30
e s age=0
```

正例：1

```java
i name = "yx" {
    s age = 30
} else {
    s age = 0
}
```

正例：2

```java
s age = $s(name = "yx" , 1 : 0)
```

1.  `if` 嵌套不宜过多，建议不超过3层。

反例：

```java
i name = "yx" d
.s state = 30
.i gender = "男" d
..s state = 20
..i age = "年轻人"
...s state = 10
e  d
.s state =0
```

正例：[用策略模式或状态模式封装](https://www.processon.com/view/5dd7deebe4b0d8a683f3e4aa#map)。

1.  多级`if else` 考虑用`$case`替换。

反例：

```java
i day =1 d
.s desc = "Monday"
e  i day =2 d
.s desc = "Tuesday"
e  i day =3 d
.s desc = "Wednesday"
```

正例：

```java
$CASE(day,
	1:"Monday",
	2:"Tuesday",
	3:"Wednesday",
	4:"Thursday",
	5:"Friday",
	6:"Saturday",
	7:"Sunday",
	:"entry error"
)
```

1.  与或逻辑运算统一使用`&&`、`||`，禁止使用其他`&`、`!`、`,`。

反例：

```java
s firstName = "Yao"
s lastNmae = "Xin"
if firstName = "Yao",lastNmae = "Xin" 
.w "true",!
e  d
.w "false",!
```

正例：

```java
	s firstName = "Yao"
	s lastNmae = "Xin"
	if (firstName = "Yao")&&(lastNmae = "Xin") {
		w "true",!
	} else {
		w "false",!
	}
```

1.  块级语法命令要全拼。例如`for`，`if`命令在块语法中不要缩写，这样语义更强。

反例：

```java
ClassMethod Login()
{
    s info =""
	f  s info = $o(^IRIS("info", info)) q:info="" d
	.s flag = $p(^IRIS("info", info), "^", 1)
	.i flag = "1" d
	..// todo 
}
```

正例：

```java
ClassMethod Login()
{
    s info = ""
	for {
		s info = $o(^IRIS("info", info))
		q:(info = "")
		s flag = $p(^IRIS("info", info), "^", 1)
		if (flag) {
			//to do 
		}
	}
}
```

1.  后置表达式要加括号,并且等号两侧加上空格。保持风格统一。

反例：

```java
q:info=""
```

正例：

```java
q:(info = "")
```

1.  **多条件后置表达式**（如 `continue:` `q:` `b:` 后的条件）：括号**内部**的条件运算符两侧**加空格**，括号与`&&`/`||`之间**不加空格**。例如：正例`)&&(`，反例`) && (`

反例：

```java
continue:(hospId'="") && (hospId'=($p(^CTLOC(locId),"^",22)))
q:(inci = "") && (arcim = "") && (phcdf = "")
```

正例：

```java
continue:(hospId '= "")&&(hospId '= ($p(^CTLOC(locId), "^", 22)))
q:(inci = "")&&(arcim = "")&&(phcdf = "")
```

**说明**：括号内的`=` `'=` 等运算符两侧加空格，但`)`与`&&`之间、`&&`与`(`之间不加空格。
